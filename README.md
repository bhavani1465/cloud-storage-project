# Cloud-Based Smart File Storage System

A mini Google Drive-like application built using Flask and AWS S3.

## Features
- File upload to AWS S3
- Auto-generated shareable download link
- Simple login page
- Secure cloud storage

## Technologies Used
- Python Flask
- HTML Templates
- AWS S3
- Boto3

## How to Run
1. Install dependencies
2. Create .env file
3. Run "python app.py"
